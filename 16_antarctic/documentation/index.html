BEFORE YOU START

Please note that this is a general guide covering the majority of features you may encounter in our Premium Bootstrap 3 Templates. However not all templates include each feature covered, thus some paragraphs of this guide might be irrelevant to the design of your particular template.

HTML Structure

This template is based on a responsive layout with many column sizes you can use. Most of them are created using the div elements. The main content is nested under the div with the id of content type: page, about, copyright etc. More details on the methods of column layouts creation please see here.

   1:  <html>
   2:      <head>
   3:          <!-- Meta tags, css links and favicon links -->
   4:      </head>
   5:      <body>
   6:          <!-- Page container for every element -->
   7:          <div id="page">
   8:              <header>
   9:                  <!-- Header of the page -->
  10:              </header>
  11:              <section id="about">
  12:                  <!-- About section -->
  13:              </section>
  14:              <section id=". . .">
  15:                  <!-- Here you can use any of presented in a template section -->
  16:              </section>
  17:              <footer>
  18:                  <!-- Footer content -->
  19:              </footer>
  20:              <div id="copyright">
  21:                  <!-- Copyright text and social icons -->
  22:              </div>
  23:          </div>
  24:          <script type="text/javascript">
  25:              // JavaScript links 
  26:          </script>
  27:      </body>
  28:  </html>

CSS Files

There is a number of CSS files included in the template. They belong to the Bootstrap framework, plugins and theme styles. The list of all css files is as follows:

flexslider.css Slider styles of FlexSlider plugin.
prettyPhoto.css Styles of prettyPhoto plugin.

style.css
This is the main stylesheet for the template which includes all styles related to the appearance of the elements you see on each page. This stylesheet code has been labeled for ease of use and all style rules are grouped under particular headings. In the lower part of the code there are media queries which are responsible for adaptability

Parallax Backgrounds

Many fullscreen background elements of the template use parallax effect when scrolling.

NOTE: The effect is disabled for touch-screen devices to improve performance.

To enable the effect for any new layout section, having first attributed it a unique id in the HTML and CSS code, find /* PARRALAX */ section header in the myscript.js file, duplicate the line jQuery('#get_in').parallax("50%", -0.5); and replace get_in value with your new section id for example _MY_NEW_ID so the code looks like this


/*-----------------------------------------------------------------------------------*/
/*	PARRALAX
/*-----------------------------------------------------------------------------------*/
   1:  jQuery(window).load(function() {
   2:     if (jQuery(window).width() > 1025){
   3:        jQuery('#get_in').parallax("50%", -0.5);
   4:        jQuery('#_MY_NEW_ID').parallax("50%", -0.5);
   5:     }
   6:  });

prettyPhoto Galleries

The template makes use of the prettyPhoto gallery javascript plugin which is simple to set up on any image element by observing the following conventions.

  • The image must be wrapped in the <a> tag and contain href that points to the large image (the one you want to show when the user clicks on the thumbnail).
  • Inside the <a> you need an img element whose src value points to the thumbnail image.
As so:

   1:  <a class="gallery_img" href="images/gallery/1_big.jpg" rel="prettyPhoto[portfolio1]">
   2:      <img src="images/gallery/1.jpg" alt="" />
   3:  </a>

Flickr

If your template design doesn't feature Flickr widget, you can add one by inserting the following code at the bottom of myscript.js file, replacing the Flickr ID 52617155@N08 with your own (a valid ID must contain the @ sign)

   1:  jQuery(document).ready(function() {
   2:  $.getJSON("http://api.flickr.com/services/feeds/photos_public.gne?id=52617155@N08&lang=en-us&format=json&jsoncallback=?", 
	function(data){
   3:        $.each(data.items, function(i,item){
   4:  		   if(i<=5){ // <â€” change this number to display more or less images
   5:              $("<img/>").attr("src", item.media.m.replace('_m', '_s')).appendTo(".FlickrImages ul")
   6:              .wrap(<li><a href='" + item.link + "' target='_blank' title='Flickr'></a></li>");
   7:           }
   8:        });
   9:     });
   10:  });

... and then inserting the following code at the bottom of the index.html file

   1:  <div class="flickrs">
   2:     <div class="FlickrImages clearfix">
   3:        <ul></ul>
   4:     </div>
   5:  </div>

Twitter

In order to have your tweets displayed in the Twitter widget, in the following code inside myscript.js file replace Yourname value with your Twitter user name. 

   1:  jQuery(document).ready(function() {
   2:     jQuery('footer .twitter_wrap').tweet({
   3:        modpath: 'twitter/',
   4:        count: 1,
   5:        username : 'Yourname'
   6:     });
   7:  });

In case you need to turn off the Twitter widget altogether, simply comment out the following code in the index.html file 

   1:  <div class="twitter_wrap">
   2:      <ul class="twitter_list tweet_module"><ul>
   3:  </div>

If your template design doesn't feature Twitter widget please download the relevant library from GitHub.

Configure Google Map

To customize the Google maps location on the contact form page

  • Go to Google maps service and select your location.
  • In the right-hand corner of the map interface click the gear-wheel icon and select Share and embed map option
  • In the resulting pop-up window select Embed map tab
  • Copy the iframe code and use it to replace the default code in the index.html file contacts section src variable 
   1:  <section id="contacts">
   2:     <div id="map">
   3:        <iframe width="100%" height="340" frameborder="0" scrolling="no" marginheight="0" marginwidth="0"
           src="http://maps.google.com/maps?f=q&source=s_q&hl=en&geocode=&q=london&sll=37.0625,-95.677068&sspn=42.631141,90.263672
&ie=UTF8&hq=&hnear=London,+United+Kingdom&ll=51.500141,-0.126257&spn=0.026448,0.039396&z=14&output=embed"></iframe>
   4:     </div>
   5:  </section>

If your template doesn't feature the contact form page, the Google maps iframe code can still be used in any section of the HTML layout.

Configure Video block

Suppose you want to display a video from YouTube or Vimeo. Once you have gotten the default embed codes for sharing, you just wrap the codes with the "video_block" div class. Please see the code below for example:

   1:  <div class="video_block">
   2:      <!-- Embed code from YouTube or Vimeo -->
   3:      <iframe src="http://www.youtube.com/embed/c9MnSeYYtYY" 
   4:          frameborder="0" allowfullscreen></iframe>
   5:  </div>

Configure Contact Form

If your template features a contact form, it will allow your website visitors to send feedback to your email address. To make the contact form functional you need to insert your destination email address into the mail.php file code replacing the dummy address youdomain@gmail.com

   1:  define("CONTACT_FORM", 'youdomain@gmail.com');// Replace this with your own email address
